# Indiana Bass'N Gals Static Replica Starter

This package is a clean static-site starter built to resemble the public Indiana Bass'N Gals homepage structure and navigation so you can edit it in Adobe Dreamweaver and publish it with GitHub Pages.

## Files

- `index.html` = homepage replica
- `about.html`, `tournaments.html`, etc. = inner-page starters matching the menu
- `assets/css/styles.css` = site styling
- `assets/js/script.js` = mobile navigation toggle

## How to use in Dreamweaver

1. Open the `indiana-bass-ngals-site` folder as a local site in Dreamweaver.
2. Edit text directly in the HTML files.
3. Replace remote image URLs with your own local images inside `assets/images`.
4. Preview in browser from Dreamweaver.
5. Commit the full folder to a GitHub repository.

## How to publish with GitHub Pages

1. Create a GitHub repository.
2. Upload all files and folders in this package.
3. In GitHub, open **Settings > Pages**.
4. Set the source to the `main` branch and the root folder.
5. Save. GitHub will publish the site automatically.

## Notes

- This is a starter replica, not a pixel-perfect export of the live Wix site.
- A few images currently reference the public site so the layout looks complete immediately.
- For long-term control, download and replace those with local images.
- GitHub Pages does not run server-side code, so forms need a third-party handler.
